## âœ¨ Fitur Utama

Aplikasi ini terdiri dari beberapa halaman utama:

1.  [cite_start]**Splash Screen:** Tampilan splash screen aplikasi selama 5 detik yang menampilkan identitas [cite: 14].
2.  [cite_start]**Dashboard:** Halaman utama dengan *Bottom Navigation* untuk berpindah antar halaman. [cite: 16]
3.  [cite_start]**Biodata:** Halaman formulir untuk input data diri (Nama, Alamat, Jurusan, Gender, Tanggal Lahir) [cite: 18] yang disimpan secara lokal menggunakan `shared_preferences`.
4.  [cite_start]**Kontak:** Halaman yang menampilkan daftar kontak statis menggunakan `ListView`. [cite: 20]
5.  [cite_start]**Kalkulator:** Kalkulator fungsional yang dapat melakukan operasi tambah, kurang, kali, bagi, kuadrat, dan akar kuadrat. [cite: 32]
6.  [cite_start]**Cuaca:** Tampilan UI statis untuk informasi cuaca di 2 kota (Bandung dan Jakarta) dengan animasi Lottie dan prakiraan 7 hari. [cite: 33]
7.  [cite_start]**Berita:** Halaman yang menampilkan daftar berita statis dalam bentuk `Card` [cite: 38] dan dapat diklik untuk melihat detailnya.
